В качестве массива значений таксономии выступает многомерный массив структуры: 
```php
[
  [
    'TITLE' => 'Катализаторы', // Текст, который выводится
    'VALUE' => '414', // Если ключ VALUE не задан, то значение берется из TITLE
    'DISABLED' => true, // Ключ, указывающий, что данный 
                        // вариант будет в цепочке, но не будет 
                        // выступать, как вариант значения
    'CHILD' => [ // Структура дочерних вариантов, 
                 // цепочка может быть до бесконечности
      ['TITLE' => 'Катализатор'],
      ['TITLE' => 'Кожух катализатора'],
      ['TITLE' => 'Крепление катализатора'],
      ['TITLE' => 'Кронштейн катализатора'],
      ['TITLE' => 'Обманка катализатора'],
      ['TITLE' => 'Прокладка катализатора'],
    ]
  ],
  ['TITLE' => 'Выхлопные трубы']
];
```
